Меню Help

  • Help Topics - при выборе этого пункта вызывается справочная система Аргус, с которой Вы в данный момент работаете.

  • Help Language - позволяет выбирать язык справочной системы Аргус.

  • Help on Help - инструкции по работе со справочной системой Аргус.

  • Argus Web Site - посетить Web-сервер поддержки системы Аргус — http://www.ritlabs.com/argus/

  • Credits - при выборе этого пункта открывается окно, в котором перечислены лица, так или иначе связанные с проектом Аргус.

  • Licence - при выборе этого пункта выводится окно с текстом лицензионного соглашения, которое Вы приняли при установке системы Аргус на Ваш компьютер.

  • About - при выборе этого пункта выводится информационное окно, содержащее сведения о версии программы, авторских правах а так же номер регистрационного ключа.

  • How To Register - при выборе этого пункта выводится информационное окно, содержащее сведения о том, как легально зарегистрировать программу Аргус.

  • Eneter Registration Code... - при выборе этого пункта выводится диалоговое окно регистрации программного продукта. Вам необходимо ввести ключ регистрации и пароль, полученные от дилера.
