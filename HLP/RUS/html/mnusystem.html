Меню System

  • Open Dialup Line - этот пункт служит для открытия окна линии почтовой системы, в раскрывающемся подменю будет предложено выбрать конкретную линию, окно которой Вы желаете открыть.

  • TCP/IP Daemon - этот пункт служит для запуска TCP/IP Daemon.

  • Interface Langaage - позволяет выбрать язык интерфейса.

  • Mirror - при выборе этого пункта создается вторая копия окна Аргуса которая в точности

  • повторяет состояние главного окна.

  • Close Window - закрывает соответствующее окно Аргуса, если это было единственное окно системы, то происходит выход из нее.

  • Exit - выход из системы. Дублируется сочетанием клавиш Alt+X.
